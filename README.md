# clicker menu 2.0

A simple clicker game built with Python and Tkinter.

## 🎮 Description

This is a beginner Python project where you click a button to increase your score.

The game has a simple upgrade system:
- Each click gives you 1 point at the start
- After reaching 200 clicks, each click gives 2 points

## ⚙️ Features

- Graphical interface (Tkinter)
- Click counter
- Upgrade system (click power increases at 200 clicks)
- Simple and lightweight project

## 🚀 How to run

1. Make sure you have Python installed (3.x)
2. Download the project
3. Run the file:

```bash
python clicker.py
